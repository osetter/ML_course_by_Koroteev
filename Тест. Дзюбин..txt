	~%0% Среднее значение выборки
	~%0% Дисперсия выборки

	~%0% Экспоненциальная функция

}

::Линейная регрессия::Коэффициент наклона (slope) в уравнении линейной регрессии предстовляет собой... {
	~%0% Среднее значение зависимой переменной
	~%0% Среднее значение независимой переменной
	~%100% Изменение зависимой переменной при изменении независимой на единицу
	~%0% Константа в уравнении линейной регрессии
}

::Множественная регрессия::Множественность в множественной регрессии означает... {
	~%0% Большее количество данных
	~%100% Использование нескольких независимых переменных
	~%0% Высший уровень статистической значимости
	~%0% Увеличение точности модели
}

::Множественная регрессия::Коэффициенты которые оцениваются в множественной регрессии... {
	~%100% Коэффициенты наклона для каждой независимой переменной
	~%0% Только константа
	~%0% Коэффициенты наклона для зависимой переменной
	~%0% Только стандартные отклонения
}

::Регрессия на реальном датасете::Факторами которые могут повлиять на точность регрессионной модели на реальном датасете являются... {
	~%0% Только размер выборки
	~%33% Качество исходных данных
	~%33% Выбор используемого алгоритма регрессии
	~%33% Наличие выбросов в данных
}

::Регрессия на реальном датасете::Качество регрессионной модели на реальном датасете можно оценить с помощью... {
	~%33% Используя только среднеквадратичную ошибку
	~%33% Путем визуализации распределения остатков
	~%0% Исключительно с помощью коэффициента детерминации
	~%33% При сравнении предсказанных значений с реальными данными
}

::Логистическая регрессия::Логистическая регрессия чаще всего применяется в задачах... {
	~%0% Предсказание цен на недвижимость
	~%0% Распознавание объектов на изображениях
	~%50% Прогнозирование вероятности бинарного события
	~%50% Классификация объектов по категориям
}

::Логистическая регрессия::Коэффициенты в логистической регрессии можно интерпретировать как... {
	~%33% Как изменение вероятности для зависимой переменной
	~%33% Как изменение среднего значения зависимой переменной
	~%33% Как абсолютные значения предсказаний модели
	~%0% Как стандартные отклонения зависимой переменной
}

::Классификация на реальных данных::Метриками которыми часто используются для оценки производительности моделей классификации являются... {
